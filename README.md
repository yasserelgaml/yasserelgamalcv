# Executive Telecom Portfolio - Yasser Elgamal

A premium, executive-level portfolio website designed for a telecom operations leader. 

## 🚀 Features
- **Multilingual Support**: Full Arabic and English integration (RTL/LTR).
- **Executive Dashboard**: Interactive KPI tracking using Recharts.
- **S.T.A.R. Case Studies**: Detailed project showcase using the Situation, Task, Action, Result methodology.
- **AI Innovation Hub**: Section dedicated to AI-driven telecom solutions.
- **Corporate Design**: Clean, modern aesthetic inspired by McKinsey and IBM.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
- **Theme**: next-themes (Dark/Light mode)

## 📦 Deployment
This project is optimized for deployment on **Vercel**.
